## 🌟 ภาพรวมโครงการ (Project Overview)
**PMP Mock Exam Simulator** เป็นเว็บแอปพลิเคชันแบบ Single Page Application (SPA) ที่พัฒนาขึ้นโดยใช้ Vanilla JavaScript, HTML5 และ CSS3 (ไม่มี Framework หรือ external library เสริม เพื่อความรวดเร็วและเบาที่สุด) สำหรับผู้ที่กำลังเตรียมตัวสอบใบอนุญาต **PMP (Project Management Professional)** 

ตัวระบบมาพร้อมคลังข้อสอบมากกว่า **500+ ข้อ** ที่ดึงมาจากสถานการณ์จริงในตำรา PMBOK และแนวข้อสอบ PMP ล่าสุด พร้อมคำอธิบายเฉลยและเหตุผลเชิงลึกเป็น**ภาษาไทย**ครบทุกข้อ

## ✨ คุณลักษณะเด่น (Key Features)

### 1. โหมดการเรียนรู้และทำข้อสอบ (Dual Testing Modes)
*   **Practice Mode (โหมดฝึกฝน):** เหมาะสำหรับการตะลุยโจทย์ทีละข้อ ระบบจะเฉลยผลลัพธ์ (ถูกต้อง/ผิดพลาด) ทันทีที่เลือกคำตอบ พร้อมแสดงบทวิเคราะห์คำอธิบายภาษาไทยโดยละเอียด
*   **Exam Mode (โหมดสอบจริง):** จำลองสภาพแวดล้อมการสอบจริง โดยระบบจะใช้ **Stratified Random Sampling** ดึงข้อสอบจากคลังทั้งหมดมาสุ่ม **185 ข้อ**, จำกัดเวลาทำข้อสอบรวม 240 นาที (มีระบบนับเวลาถอยหลังรวมและระบบส่งกระดาษคำตอบอัตโนมัติเมื่อหมดเวลา), ซ่อนเฉลยและการแจ้งเตือนทั้งหมด และสรุปคะแนนวิเคราะห์ผลอย่างละเอียดเมื่อส่งกระดาษคำตอบ โดยครอบคลุมเนื้อหาตามโครงสร้าง PMP เสมือนจริง:
    *   **People (33%):** สุ่มมา 61 ข้อ เน้นทักษะความเป็นผู้นำ การจัดการทีม และผู้มีส่วนได้ส่วนเสีย
    *   **Process (41%):** สุ่มมา 76 ข้อ เน้นเทคนิคการบริหารโครงการ ทั้งแบบ Predictive และ Agile
    *   **Business Environment (26%):** สุ่มมา 48 ข้อ เน้นการทำโครงการให้สอดคล้องกับกลยุทธ์องค์กร และสร้างมูลค่า (Business Value)
    *   *(ในแต่ละหมวดหมู่จะจัดสัดส่วนแนวทางเป็น: Predictive ~40% | Adaptive/Hybrid ~60%)*

### 2. ระบบแบ่งชุดข้อสอบละ 60 ข้อ (Flexible Question Sets)
*   สำหรับการทำข้อสอบที่ใช้เวลาพักเบรก ระบบจึงมีฟังก์ชันในการแบ่งชุดข้อสอบเป็น**เซ็ตละ 60 ข้อ** เพื่อให้ผู้สอบสามารถทยอยฝึกฝนและทำความคุ้นเคยทีละส่วนได้อย่างเป็นระบบ

### 3. แผงควบคุมข้อสอบอัจฉริยะ (Interactive Control Panel)
*   **Per-Question Timer:** กำหนดเวลาทำข้อสอบต่อข้อได้ (ค่าเริ่มต้น 75 วินาที) พร้อมตัวเลือกสำหรับล็อคข้อสอบและปัดเป็นข้อผิดทันทีที่หมดเวลา (Timeout Lock) เพื่อฝึกทักษะการบริหารเวลา
*   **Shuffle Questions:** ฟังก์ชันสุ่มลำดับข้อสอบเพื่อไม่ให้เกิดการจำตำแหน่งข้อเดิม
*   **Flag / Pin for Review:** ปักหมุดทบทวนข้อที่ยังไม่มั่นใจ เพื่อกลับมาทำหรือตรวจสอบในภายหลังได้ง่าย
*   **Navigation Grid Panel:** แผงนำทางขวาแสดงเลขข้อสอบ (1 ถึง N) พร้อมระบุสถานะสีชัดเจน:
    *   *สีเขียว:* ตอบแล้ว
    *   *สีส้ม/หมุด:* ปักหมุดทบทวน
    *   *สีเทาเข้ม:* ยังไม่ได้ตอบ
    *   *ขอบสีสว่าง:* ข้อปัจจุบันที่กำลังเปิดทำ

### 4. ระบบสรุปผลลัพธ์และทบทวนข้อผิดพลาด (Comprehensive Review Dashboard)
*   เมื่อทำข้อสอบเสร็จ ระบบจะนำเสนอรายงานสรุปผลลัพธ์แบบ visual รัศมีวงกลมแสดงเปอร์เซ็นต์คะแนน
*   แสดงสถิติเวลาที่ใช้เฉลี่ยต่อข้อ (Average Time spent), จำนวนข้อที่ถูก, ผิด และข้าม
*   **Interactive Reviews Area:** รายการแสดงข้อสอบทั้งหมดหลังสอบเสร็จ สามารถเลือกฟิลเตอร์เฉพาะข้อที่ **"ตอบผิด (Incorrect)"**, **"ตอบถูก (Correct)"** หรือ **"ปักหมุดไว้ (Flagged)"** เพื่อความสะดวกในการทบทวนจุดอ่อนของตัวเอง

### 5. การออกแบบและเทคโนโลยี (Premium UI & Sound System)
*   **Modern Dark Theme & Glassmorphism:** หน้าจอสวยงาม ดูพรีเมียม สบายตา เหมาะสำหรับใช้จ้องหน้าจอทำข้อสอบเป็นเวลานาน
*   **Web Audio API Synth Sounds:** ใช้ระบบจำลองเสียงสังเคราะห์ (Beep) ผ่าน AudioContext ของเบราว์เซอร์เพื่อแจ้งเตือนจังหวะการกดเสียงสั้น ตอบถูก ตอบผิด หรือเตือนเวลาใกล้หมด โดยไม่ต้องดาวน์โหลดไฟล์เสียงจากภายนอก

## 📂 โครงสร้างโฟลเดอร์ (Folder Structure)

```bash
exam_test_app/
├── index.html       # โครงสร้างหน้าหลักและ UI ทุกส่วน (Launcher, Quiz Arena, Results)
├── style.css        # สไตล์การแสดงผล Dark Theme, Glassmorphic panels, responsive layouts
├── app.js           # หัวใจหลักของ Logic ควบคุม State, การจับเวลา, และปฏิสัมพันธ์ต่างๆ
├── questions.json   # คลังข้อสอบทั้งหมดในรูปแบบ JSON (500+ ข้อ)
└── README.md        # เอกสารแนะนำการใช้งานโครงการนี้
```

## 🚀 วิธีการเข้าใช้งาน (How to Use)

### ใช้งานผ่านระบบออนไลน์ (GitHub Pages)
สามารถเข้าใช้งานได้ทันทีโดยไม่ต้องติดตั้งซอฟต์แวร์ใดๆ ผ่านเบราว์เซอร์ที่ลิงก์:
👉 **[https://sleepingsong.github.io/exam_test_app/](https://sleepingsong.github.io/exam_test_app/)**

### รันบนเครื่องคอมพิวเตอร์ส่วนตัว (Local Execution)
1.  ทำการดาวน์โหลด หรือ clone repository นี้ลงบนเครื่องของคุณ:
    ```bash
    git clone https://github.com/Sleepingsong/exam_test_app.git
    ```
2.  ดับเบิลคลิกเปิดไฟล์ `index.html` บนเว็บเบราว์เซอร์ใดก็ได้ (Chrome, Firefox, Safari, Edge)
3.  *หรือ* เพื่อป้องกันปัญหา CORS Policy ของเบราว์เซอร์ในการโหลดไฟล์ JSON ท้องถิ่น แนะนำให้รันผ่าน local server เช่น:
    *   **VS Code:** ใช้ extension `Live Server`
    *   **Python:** รันคำสั่ง `python -m http.server 8000` ในโฟลเดอร์โครงการ แล้วเข้าใช้งานผ่าน `http://localhost:8000`
    *   **Node.js:** รันคำสั่ง `npx serve` ในโฟลเดอร์โครงการ

## 🛠️ โครงสร้างไฟล์ข้อสอบ (Questions Schema)

ข้อสอบถูกบันทึกไว้ในไฟล์ `questions.json` โดยแต่ละข้อประกอบด้วยฟิลด์ต่างๆ ดังนี้:

```json
[
  {
    "id": 1,
    "file_name": "image_reference.png",
    "question": "คำถามภาษาอังกฤษจำลองสถานการณ์การบริหารโครงการ...",
    "options": [
      "a. ตัวเลือกข้อ A",
      "b. ตัวเลือกข้อ B",
      "c. ตัวเลือกข้อ C",
      "d. ตัวเลือกข้อ D"
    ],
    "correct_answer": "c",
    "explanation": "ตัวเลือก c ถูกต้อง: อธิบายรายละเอียดวิเคราะห์ตัวเลือกเชิงเปรียบเทียบในแง่ PMBOK6/7..."
  }
]
```

หากต้องการเพิ่มหรือแก้ไขโจทย์ สามารถจัดการเพิ่มข้อมูลลงในไฟล์ `questions.json` นี้ได้โดยตรง โครงสร้างจะถูกดึงเข้าสู่ระบบและอัปเดตสถิติตัวเลขในหน้าควบคุม (Launcher) โดยอัตโนมัติ
